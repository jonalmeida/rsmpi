Point to point communication

Endpoints of communication are mostly described by types that implement the Source and Destination trait. Communication operations are implemented as traits that have blanket implementations for these two traits.

Unfinished features

  • 3.2.6: MPI_STATUS_IGNORE
  • 3.6: Buffer usage, MPI_Buffer_attach(), MPI_Buffer_detach()
  • 3.7: Nonblocking mode:
    • Completion, MPI_Waitany(), MPI_Waitall(), MPI_Waitsome(), MPI_Testany(), MPI_Testall(), MPI_Testsome(), MPI_Request_get_status()
  • 3.8:
    • Cancellation, MPI_Cancel(), MPI_Test_cancelled()
  • 3.9: Persistent requests, MPI_Send_init(), MPI_Bsend_init(), MPI_Ssend_init(), MPI_Rsend_init(), MPI_Recv_init(), MPI_Start(), MPI_Startall()
